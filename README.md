# Moonlit Mess

A quiet emotional space between sky and sea.

## Deploy to GitHub Pages

1. Create a repo named `moonlit-mess`
2. Push all files to `main` branch root
3. Settings → Pages → Source: main / root
4. Live at `https://yourusername.github.io/moonlit-mess`

## Files

```
index.html   — structure and all screens
style.css    — full CSS: scene + UI + animations
scene.js     — lunar phase, moon shadow, shooting stars (no canvas)
music.js     — ambient piano + ocean soundscape
lantern.js   — release ritual animation
app.js       — flow logic, Claude API, message pool
```

## Background

No canvas. No particle systems. No heavy layers.
The scene is pure CSS: sky gradient, static SVG stars,
CSS-animated moon glow, CSS moonpath reflection, CSS ripples.
The moon phase updates daily from real astronomical data.
